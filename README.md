# Book CRUD Application

A simple book tracking application with PHP backend and vanilla JavaScript frontend.

## Setup Instructions

1. **Start XAMPP**: Make sure Apache and MySQL are running in XAMPP control panel.

2. **Create Database**:
   - Open phpMyAdmin (http://localhost/phpmyadmin)
   - Create a new database named `book_tracker`
   - Import the `database_setup.sql` file from this project

3. **Access the Application**:
   - Frontend: http://localhost/y3/book-crud/frontend/
   - API: http://localhost/y3/book-crud/backend/api.php

## Features

- Add, edit, delete books
- Track reading progress
- Filter books by status
- Dashboard with statistics
- Reading session logging
